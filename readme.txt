Codi font dels 2 components principals del projecte Agenda Oberta,
desenvolupat gràcies a l'ajut de la Fundació punt CAT

Plataforma basada en Yii. (arxius de Yii no inclosos) http://www.yiiframework.com/

Descripció dels arxius:

* /beta
Agregador de fonts d'informació en format xml, rss, html, ical i google calendar

* /api
Api que permet reutilitzar els continguts

* /bd
Dump de la base de dades
